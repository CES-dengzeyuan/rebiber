# Rebiber-Webdemo: the web wrapper of the Rebiber tool.

### Installation

```
pip install -r requirements.txt --user
```

### To run a Flask App
```
python -m flask run
```

### To activate the debug mode
```
FLASK_DEBUG=1 flask run
```
